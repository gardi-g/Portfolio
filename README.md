# Portfolio created in ReactJS

# Installation 
  1. Clone the repo 
   ```sh
   git clone https://github.com/gardi-g/Coding-Challenge-Torqata-
   ```
  2. If you don't have Node.js https://nodejs.org/en/download/
 
  3. Install NPM packages
   ```sh
   npm install
   ```
   4. To run project
   ```sh
   npm start
   ```
